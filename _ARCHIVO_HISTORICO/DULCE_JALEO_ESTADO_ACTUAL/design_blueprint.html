🏆 Reglas de Oro (Innegociables)

1. IDs Intocables
Nunca cambies `scanner-tab`, `agendaList` o `saveContact`. Romperás la conexión con Airtable.
2. CSS Semántico
Prohibido `background: red;`. Usa siempre `var(--rose)` o `var(--gold)`.
3. Iconografía Lucide
Usa `data-lucide`. Standard: `stroke-width: 2`, `size: 24px`.
4. Layout Fluido
Nada de `width: 400px;`. Siempre `%`, `vh` o `flex/grid`.

🎨 Definición de Color (Luxury HSL)

ORO DULCE - `#D4AF37` (Acentos y Títulos)
NEGRO PITCH - `#0a0a0a` (Fondo de Inmersión)
CRISTAL LUX - `rgba(255,255,255,0.05)` (Paneles Glass)

🛠️ Catálogo de Componentes Reales (Fase 2 & 3)

Basado en tus referencias. Estos son los módulos que puedo configurar con exactitud:

1. Top Bar Luxury Config
• Izquierda: Avatar circular + Saludo ("Hola, Alex").
• Centro/Derecha: Buscador tipo cápsula (Lupa integrada).
• Extremo Derecho: Icono de acción (Ajustes o Notificaciones).
2. Hero Summary (Impacto)
• Tipografía: Números extra-grandes (ej: 3.540,00 €).
• Micro-Info: Sub-indicador de estado (ej: "Default" o "En curso").
• Botón Acción: Botón tipo cápsula para acceso rápido (ej: "Ver Facturas").
3. Action Matrix (Iconos Rápidos)
• Diseño: Fila de botones circulares con fondo `glass-mild`.
• Feedback: Animación de escala al pulsar (Haptic feeling).
• Etiquetas: Texto corto inferior para máxima claridad.
4. Smart List Cards
• Icono Lateral: Icono con color de estado (Verde=Ok, Rojo=Alerta).
• Multi-línea: Título principal + Fecha/Categoría debajo.
• Valor Derecha: Importe o estado alineado a la derecha.
